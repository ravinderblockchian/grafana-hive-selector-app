# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-28

### Added
- Initial release of Grafana Hive Selector App Plugin
- Nested tree structure for browsing hive groups, sub-groups, and nodes
- Search functionality across all hierarchy levels
- Auto-injection into Grafana dashboard control bar
- Toggle button with eye icon to show/hide the selector
- Support for Grafana >= 10.4.0

### Features
- Hierarchical navigation through hive structure
- Intelligent search prioritization (groups, then sub-groups, then nodes)
- Seamless integration with Grafana dashboards
- Automatic appearance on dashboard pages
